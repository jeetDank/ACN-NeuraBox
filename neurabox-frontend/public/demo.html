Easy Integration Guide - Embed the AI Chat Widget on Any Website

How to Integrate

Step 1: Copy the Embed Code

Add this iframe code to your HTML page where you want the widget to appear:

<iframe id="neurabot-widget" src="http://localhost:3000/widget" width="100%" height="600px" style="border: none; border-radius: 8px;"> </iframe>

Step 2: Configuration

  • src: Point to your NeuraBot widget URL
  • width: Adjust as needed (100% fills container)
  • height: Minimum 600px recommended

Step 3: For Production

src="https://yourdomain.com/widget"
