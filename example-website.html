How It Works

The NeuraBot widget on the right uses <iframe> to embed the chat interface. When visitors click the button, they can ask questions and get instant answers powered by your knowledge base.

Integration Code

To add this widget to your website, simply include this HTML code:

<!-- NeuraBox Widget --> <iframe id="neurabot-widget" src="http://localhost:3000/widget" width="100%" height="600px" frameborder="0" allowfullscreen> </iframe>

Configuration

For production, update the iframe src to point to your deployed instance:

  • Development: http://localhost:3000/widget
  • Production: https://yourdomain.com/widget

The widget communicates with your backend API to provide intelligent responses based on your knowledge base.
